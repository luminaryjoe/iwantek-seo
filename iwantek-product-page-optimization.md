## 8. 页面速度优化

### 关键优化项

1. **图片优化**
   - 使用 WebP 格式
   - 实施懒加载
   - 响应式图片 (srcset)

2. **代码优化**
   - 压缩 CSS/JS
   - 延迟加载非关键 JS
   - 移除未使用的代码

3. **缓存策略**
   - 浏览器缓存静态资源
   - CDN 加速

4. **目标指标**
   - First Contentful Paint: < 1.8s
   - Largest Contentful Paint: < 2.5s
   - Time to Interactive: < 3.8s
